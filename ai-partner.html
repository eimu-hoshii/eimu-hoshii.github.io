AI STREAM PARTNER

ID: 01
Gemini API Python Twitch API VOICEVOX

PROJECT OVERVIEW

配信者の発言をリアルタイムで聞き取り、AIキャラクターが相槌や返答をしてくれる配信補助ツールです。
RVC（Voice Changer）との連携を前提としており、「架空の相棒」 との掛け合いを実現します。

FEATURES

1. リアルタイム応答
マイク入力を自動検知し、思考から発話まで全自動で行います。

2. 個性的なキャラクター設定
システムプロンプトにより、性格付けが行えます。
Geminiの高度なコンテキスト理解により、文脈を踏まえた自然な会話が可能です。

3. RVC完全連携
AI音声を仮想オーディオデバイスに出力し、ボイスチェンジャーを通すことができます。

DEVELOPMENT NOTE

最大の課題は「遅延（レイテンシ）」の削減でした。
APIレスポンスの高速化と、音声生成のパイプライン処理を並列化することで、会話のテンポを損なわない速度を実現しました。
